Class 3 - Basic Github

✔️For 1st repository

  1. At first login to Github.
  2. Then press the (+) button to add new repository.
  3. Make SURE the name of the repository is the SAME as the folder you are gonna make.
  4. Then add the code provided by Github one by one.
  5. git init is used to initialize. Once per project is enough.
  6. Use git add . to add the file. Must be used for every new file
  7. Then paste the rest of the code as it is. But change the commit to a new one.
  8. If it's your first time, you may need to add your email and github User name because after commit you will get a fatal error if it is your first time.

✔️The Commands that you NEED to REPEAT while UPDATING on Github EVERYTIME!

  1. git add . -->(used to stage the commit)
  2. git commit -m "ANYTHING YOU WANT TO WRITE" -->(used to take snapshot to save to local repo)
  3. git push -->(pushes the commit in remote repo)

✔️How to deploy Your landing Page!

  1. At first go to settings
  2. Go to pages
  3. Set branches to main
  4. Then deploy and wait. You will probably get a link!
